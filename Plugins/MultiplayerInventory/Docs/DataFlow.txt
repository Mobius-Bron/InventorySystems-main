# MultiplayerInventory 插件 - 数据运行流文档

## 流程一: 玩家拾取物品

触发: 玩家按下 MainInteract 键
|
1. 按键输入
   UMIS_InventoryComponent::SetupPlayerInput() 中绑定了 PrimaryInteractAction
   → 调用 UMIS_InventoryComponent::PrimaryInteract()
|
2. 射线检测 (每帧自动运行)
   UMIS_InventoryComponent::TickComponent()
   → TraceForItem()
      从屏幕中心发射 LineTraceSingleByChannel 射线
      → 命中 Actor 的 UMIS_ItemComponent 组件
      → 命中高亮 Actor → IMIS_Highlightable::Highlight()
      → 广播 OnPickupMessage(拾取提示文本)
      → 离开时 → IMIS_Highlightable::UnHighlight()
      → 广播 OnHidePickupMessage()
|
3. 拾取物品
   PrimaryInteract() → TryAddItem(ItemComponent)
   → 获取 ItemComponent 的 ItemManifest
   → 判断是否 Stackable (检查 FMIS_StackableFragment)
   → 如果可堆叠且库存已有同类型:
       → 计算可堆叠数量
       → 广播 OnStackChange (通知 UI)
       → Server_AddStacksToItem() (RPC → 服务端)
   → 如果不可堆叠或库存无同类型:
       → Server_AddNewItem() (RPC → 服务端)
|
4. 服务端处理 (Server_AddNewItem)
   → InventoryList.AddEntry(ItemComponent)
   → ItemComponent->GetItemManifest().Manifest() — 创建 UMIS_InventoryItem
   → Item->SetTotalStackCount(StackCount)
   → FastArray 自动复制到所有客户端
   → PreReplicatedAdd → OnItemAdded.Broadcast()
   → ItemComponent->PickedUp() — 销毁世界中的 Actor
|
5. UI 更新
   UMIS_InventoryGrid 订阅了 OnItemAdded
   → UMIS_InventoryGrid::AddItem()
   → HasRoomForItem() — 空间查询
   → AddItemToIndices() — 逐个槽位:
       CreateSlottedItem() — 创建物品图标 Widget
       AddSlottedItemToCanvas() — 放置到 CanvasPanel
       UpdateGridSlots() — 更新 GridSlot 状态 (占用/堆叠数)


## 流程二: 玩家拖拽物品

触发: 玩家在库存界面对物品左键按下
|
1. 点击检测
   UMIS_SlottedItem::NativeOnMouseButtonDown()
   → OnSlottedItemClicked.Broadcast(GridIndex, MouseEvent)
   → UMIS_InventoryGrid::OnSlottedItemClicked()
|
2. 拾起物品 (PickUp)
   → 如果没有 HoverItem 且是左键:
       PickUp(ClickedInventoryItem, GridIndex)
         → AssignHoverItem(Item, GridIndex, PreviousIndex)
            设置图标、尺寸、堆叠数到 HoverItem
            SetMouseCursorWidget(HoverItem) — 鼠标跟随
         → RemoveItemFromGrid(Item, GridIndex)
            清除所有占用格子、删除 SlottedItem
|
3. 拖拽中 (每帧 Tick)
   → UpdateTileParameters(CanvasPosition, MousePosition)
     计算鼠标所在网格坐标
   → OnTileParametersUpdated()
     → CheckHoverPosition() — 检测目标位置:
        - 有无空位?
        - 只有单个物品挡路? (可交换)
        - 多个物品挡路? (不可放置)
     → 更新高亮: 绿色(可放) / 灰色(交换) / 无高亮(不可放)
|
4. 放下物品
   左键点击已有物品:
     → 如果是同类可堆叠:
         SwapStackCounts / ConsumeHoverItemStacks / FillInStack
     → 如果是不同物品且可交换:
         SwapWithHoverItem()
   在空白区域点击鼠标:
     → UMIS_InventoryWidget::NativeOnMouseButtonDown()
     → DropItem() — 放置 HoverItem 到当前位置
   在背包外区域放下:
     → 原路径: InventoryComponent->RequestDropItem() (丢弃)
     → 修改后: PutHoverItemBack() (放回原位)


## 流程三: 玩家装备物品

触发: 拖起装备物品,点击装备栏槽位
|
1. 装备槽位点击
   UMIS_EquippedGridSlot::NativeOnMouseButtonDown()
   → EquippedGridSlotClicked.Broadcast(this, EquipmentTypeTag)
   → UMIS_InventoryWidget::EquippedGridSlotClicked()
|
2. 校验是否可以装备
   CanEquipHoverItem(EquippedGridSlot, EquipmentTypeTag)
     → 槽位是否为空?
     → HoverItem 是否非堆叠?
     → HoverItem 是否有 EquipmentFragment?
     → EquipmentType 是否匹配槽位的 EquipmentTypeTag?
|
3. 创建装备图标
   EquippedGridSlot->OnItemEquipped(Item, EquipmentTag, TileSize)
     → 从 Item 获取 GridFragment + ImageFragment
     → CreateWidget<UMIS_EquippedSlottedItem>()
     → 设置图标、尺寸、叠加到 Overlay_Root
|
4. 服务端通信
   InventoryComponent->RequestEquipItem(HoverItem, nullptr)
     → Server_EquipSlotClicked(ItemToEquip, ItemToUnequip)
     → Multicast → OnItemEquipped.Broadcast(Item)
     → OnItemUnequipped.Broadcast(nullptr)
|
5. 装备组件响应
   UMIS_EquipmentComponent::OnItemEquipped(Item)
     → 获取 FMIS_EquipmentFragment
     → EquipmentFragment->OnEquip(PC)
         遍历 EquipModifiers → 执行所有装备效果
     → SpawnAttachedActor(AttachMesh)
         在世界中生成 AInv_EquipActor
         附着到角色骨骼的 SocketAttachPoint
|
6. HoverItem 清除
   InventoryGrid->ClearHoverItem()


## 流程四: 玩家卸下装备

触发: 点击装备栏上已装备的物品图标
|
1. 点击检测
   UMIS_EquippedSlottedItem::NativeOnMouseButtonDown()
   → OnEquippedSlottedItemClicked.Broadcast(this)
   → UMIS_InventoryWidget::EquippedSlottedItemClicked()
|
2. 卸下处理
   → 获取已装备物品 → AssignHoverItem (放回鼠标)
   → ClearSlotOfItem(EquippedGridSlot) — 清除槽位
   → RemoveEquippedSlottedItem(SlottedItem) — 移除图标
|
3. 服务端通信
   BroadcastSlotClickedDelegates(nullptr, ItemToUnequip)
     → RequestEquipItem(nullptr, ItemToUnequip)
     → OnItemUnequipped.Broadcast(Item)
|
4. 装备组件响应
   UMIS_EquipmentComponent::OnItemUnequipped(Item)
     → FMIS_EquipmentFragment->OnUnequip(PC)
         遍历 EquipModifiers → 撤销所有装备效果
     → RemoveEquippedActor(EquipmentType)
         销毁 3D 装备 Actor


## 流程五: 玩家消耗物品

触发: 右键物品 → 弹出菜单 → 点击消耗按钮
|
1. 右键弹出菜单
   → UMIS_InventoryGrid::CreateItemPopUp(GridIndex)
     → 可堆叠 → 显示拆分滑块
     → 可消耗 → 显示消耗按钮
     → 始终显示丢弃按钮
|
2. 消耗执行
   → UMIS_ItemPopUp::ConsumeButtonClicked()
   → OnConsume.ExecuteIfBound(GridIndex)
   → UMIS_InventoryGrid::OnPopUpMenuConsume()
   → InventoryComponent->RequestConsumeItem(Item)
   → Server_ConsumeItem(Item) (RPC)
     → TotalStackCount -= 1
     → 如果堆叠归零 → InventoryList.RemoveEntry()
     → FMIS_ConsumableFragment->OnConsume(PC)
         遍历 ConsumeModifiers → 执行所有消耗效果


## 流程六: 打开/关闭库存界面

触发: 按 ToggleInventory 键
|
1. 按键触发
   UMIS_InventoryComponent::RequestToggleInventory()
   → OnInventoryToggledRequest.Broadcast()
|
2. 外部系统响应 (由项目代码自行实现)
   监听 OnInventoryToggledRequest 委托:
     → 如果界面关闭 → Widget->OpenInventory()
         SetVisibility(Visible)
         SetInputMode(GameAndUI) + ShowMouseCursor
     → 如果界面打开 → Widget->CloseInventory()
         SetVisibility(Collapsed)
         Grid->OnHide() — PutHoverItemBack()
         SetInputMode(GameOnly) + HideMouseCursor


## 数据流图示

  [玩家输入] → [HoverItem/GridSlot点击]
       ↓
  [InventoryWidget] → [InventoryGrid]     ← UI 层
       ↓                   ↓
  [Request 命令] ────→ [InventoryComponent] ← ViewModel
       ↓                   ↓
  [Server RPC] ──────→ [服务端 InventoryComponent]
       ↓                   ↓
  [FastArray 复制] ←─→ [客户端 InventoryList]
       ↓
  [PreReplicatedAdd/Remove]
       ↓
  [OnItemAdded/Removed] → [Grid.AddItem/RemoveItem]  ← UI 自动更新
